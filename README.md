## About  
**Pixify** is a full-stack social media application inspired by Instagram.  
It allows users to connect, share moments, and interact in real-time with a beautiful, responsive UI and secure backend.

---

## Key Features 🚀  

### 🔐 Authentication & Security
- JWT Authentication  
- OTP Verification via Email  
- Password Reset with secure tokens  

### 👤 User Profile
- Update profile picture & bio  
- Follow / Unfollow users  
- View followers & following  

### 🖼️ Posts & Media
- Create & delete posts  
- Like & comment on posts  
- Save / Unsave posts  
- Image upload with Cloudinary  

### 📩 Social Interaction
- Real-time feed updates  
- User suggestions  
- Post engagement tracking  

### 📱 UI & Performance
- Fully responsive (Mobile, Tablet, Desktop)  
- Clean & modern UI  
- Optimized API calls  

---

## Tech Stack 🛠️  

### Frontend
- React.js  
- Tailwind CSS  
- Redux Toolkit  

### Backend
- Node.js  
- Express.js  
- MongoDB  
- JWT Authentication  

### Services & Deployment
- Cloudinary (Image Upload)  
- Resend / Email Service (OTP & Emails)  
- Frontend: **Vercel**  
- Backend: **Render**

---

## How to Use 🧑‍💻  

1. Open the [Pixify Live App](https://pixify-kappa.vercel.app)  
2. Register or login using OTP verification  
3. Create posts & upload images  
4. Like, comment, follow users  
5. Customize your profile
